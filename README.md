# 🤖 Personalized Gemini AI Assistant

A high-performance FastAPI backend that integrates Google's **Gemini 1.5 Flash** to provide a personalized AI experience based on user-defined profiles.

## 🚀 Features
* **Custom Personalities:** Adapts its tone, depth, and communication style based on a 10-point user profile.
* **FastAPI Backend:** Built for speed and easy integration with mobile or web frontends.
* **Secure Configuration:** Uses environment variables (`.env`) to keep API keys safe.

## 📂 Project Structure
* `newfile.py`: The main FastAPI server and AI logic.
* `requirements.txt`: List of Python dependencies.
* `.gitignore`: Ensures sensitive data (like API keys) isn't public.
